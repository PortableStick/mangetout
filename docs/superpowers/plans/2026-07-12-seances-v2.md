# Séances v2 — métriques, feedback scientifique & coach visuel — Plan d'implémentation

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development (recommended) ou
> superpowers:executing-plans. Steps en checkbox (`- [ ]`).

**Goal:** Rendre les séances utilisables (détail/édition/statut/provenance/duplication), supporter des
métriques d'équipement flexibles (rameur/tapis/…), donner un feedback scientifique transparent, corriger le
clavier du coach, et ajouter un coach visuel (3D dé-risqué par POC + guidage audio).

**Architecture:** Offline-first inchangé (writes via `getSyncManager().enqueue`, soft-delete, `sync_records`
JSON générique, nouveaux champs additifs). Métriques = catalogue de champs FERMÉ + presets. Feedback =
module pur d'heuristiques sourcées. Coach visuel = abstraction `WorkoutPlayer` avec rendus interchangeables ;
3D via `expo-gl`+`three` seulement si POC concluant.

**Tech Stack:** Expo 57 / RN 0.86 / React 19, expo-router, drizzle+expo-sqlite, TanStack Query, zod, Hono
(serveur), PocketBase. Tests : jest 29 (app, globals via `@jest/globals`), vitest (serveur).

## Global Constraints
- **Expo SDK 57 / RN 0.86 / React 19** — ne pas changer. eslint 9, jest 29 (app), vitest (serveur).
- **M16→M19 : zéro nouvelle dépendance.** **M20 uniquement** : `expo-gl`, `three` (+ loader GLTF),
  `expo-av`, `expo-speech` — versions compatibles Expo 57, à vérifier avant install.
- Offline-first : writes via `getSyncManager().enqueue(collection,'upsert',record)`, soft-delete
  `deleted:true`, `clientUpdatedAt = Date.now()`, `user` = userId.
- **Champs additifs + rétro-compat** : mapping repository fournit des défauts pour anciens records
  (`status:'done'`, `source:'manual'`, `metricSet:'strength'`).
- **Owner-scoping coach** : `user` = utilisateur vérifié serveur, jamais le modèle ; ids d'outils `^[a-z0-9]{15}$`.
- **Feedback = heuristiques transparentes sourcées, JAMAIS de prédiction individuelle.** Chaque reco porte
  `source: 'acsm'|'issn'|'nih'|'rp_heuristic'|'user_override'`.
- Kit `@/components/ui/*` + `useTheme()`, aucune couleur en dur. Langue FR, unités métriques.
- Commits Conventional Commits FR. Un milestone n'est « fait » que **vert** (typecheck+lint+tests) avant merge.
- Commandes : app `cd app && npm run typecheck && npm run lint && npm test` · serveur `cd server && npm run typecheck && npm test`.

## M19 — Feedback scientifique (heuristiques transparentes)

### Task 19.1 : Module d'heuristiques `coaching.ts` (pur, testé)
**Files:** Create `app/src/features/stats/coaching.ts`, `app/src/features/stats/coaching.test.ts`
**Produces:**
```ts
export type RecoLevel = 'good'|'low'|'high'|'info';
export type RecoSource = 'acsm'|'issn'|'nih'|'rp_heuristic'|'user_override';
export interface Reco { id: string; level: RecoLevel; message: string; source: RecoSource }
export type GoalMode = 'gain'|'cut'|'maintain';
export function weeklySetsByMuscle(workouts: {status:string}[], exercises: {muscleGroups?:string[]}[]): Record<string, number>;
export function volumeRecos(setsByMuscle: Record<string, number>): Reco[];      // ACSM ≥10 séries/sem/muscle
export function proteinTarget(bodyweightKg: number, mode: GoalMode): { min: number; max: number };  // ISSN
export function proteinReco(intake_g: number, bodyweightKg: number, mode: GoalMode): Reco;
export function calorieReco(intake_kcal: number, goal_kcal: number|undefined, mode: GoalMode): Reco;  // ~500 kcal (nih)
export function weightTrend(entries: { date: string; weight_kg: number }[]): { avg7: number|null; deltaPerWeek: number|null };
```
Règles (sourcées) : volume < 10 → `low` (acsm) ; 10–20 → `good` ; >20 → `info` (rp_heuristic « proche MRV »).
Protéines ISSN : mode `gain|maintain`→1,4–2,0 g/kg ; `cut`→2,3–3,1 g/kg. `proteinReco` : intake < min →
`low` (issn). Calories : `cut`→viser ~-500 vs maintien ; `gain`→+250..+500 ; compare à `goal_kcal` si fourni,
sinon message informatif (nih). `weightTrend` : moyenne 7 jours + delta kg/semaine, **jamais** de date projetée.
**Interdit** : toute fonction renvoyant une date/objectif atteint ou un gain projeté.
